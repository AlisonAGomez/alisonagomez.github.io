# Como adicionar novas automações

1. Crie uma nova pasta em `assets/automations/<id-da-automacao>/` se quiser anexar arquivos.
2. Adicione preview em `assets/automations/previews/<arquivo>.png`.
3. Adicione uma entrada em `assets/automations/automations.json`.
4. Use `automation/?id=<id-da-automacao>` para abrir a página dinâmica.

Quando você subir o script no GitHub, atualize o campo `repository`.
